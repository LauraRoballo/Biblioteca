# 📚 Sistema de Gestión Bibliotecaria - Liceo Femenino Mercedes Nariño

Plataforma integral diseñada para modernizar y optimizar la administración de recursos bibliográficos del Colegio Liceo Femenino Mercedes Nariño. El sistema automatiza el ciclo completo de préstamo, devoluciones y reservas, eliminando procesos manuales y mejorando la experiencia de la comunidad educativa. 🏛️📚

## 🛠️ Tecnologías Propuestas

Para el desarrollo de este sistema se han seleccionado las siguientes herramientas:

* **Lenguaje de Programación:** Java
* **Framework de Backend:** Spring Boot
* **Base de Datos:** MySQL
* **Frontend:** HTML5, CSS3 y JavaScript

## ✨ Características Principales
* **Gestión de Préstamos:** Control de fechas y disponibilidad en tiempo real.
* **Perfiles Diferenciados:** Acceso para bibliotecarios, administrativos, docentes y alumnos.
* **Reportes Estadísticos:** Generación de métricas sobre los libros más solicitados.
